# Welcome Home
Hi! My name is Yuriy Zhdanov. This is my home directory. Ubuntu 20.04.

## Installation

### Dotfiles
```bash
sudo apt install -y git
cd $HOME
git init
git remote add origin https://github.com/zhdanov/home
git fetch origin
git checkout -b main origin/main
```

### Minikube (optional)
```bash
./setup/setup.bash
```
